# 二、嵌入式 / RTOS 特定含义（核心重点）

> 这一部分才是你看 STM32 / FreeRTOS 代码的关键

------

# 1️⃣ RTOS任务系统（Task）

| 单词       | 嵌入式含义               |
| ---------- | ------------------------ |
| task       | 任务（线程级执行单元）   |
| thread     | 线程（RTOS任务等价）     |
| handle     | 任务句柄（指针）         |
| priority   | 优先级（决定抢占顺序）   |
| stack      | 任务栈                   |
| stack size | 栈大小                   |
| idle task  | 空闲任务（最低优先级）   |
| tick       | 系统节拍（RTOS时间基准） |

------

## 🧠 任务状态（非常重要）

| 状态      | 含义                     |
| --------- | ------------------------ |
| ready     | 就绪（等CPU）            |
| running   | 正在运行                 |
| blocked   | 阻塞（等事件/延时/队列） |
| suspended | 挂起（手动暂停）         |

------

## 🔧 RTOS任务API语义

| API                 | 含义       |
| ------------------- | ---------- |
| xTaskCreate         | 创建任务   |
| vTaskDelete         | 删除任务   |
| vTaskDelay          | 阻塞延时   |
| vTaskDelayUntil     | 周期运行   |
| vTaskSuspend        | 挂起任务   |
| vTaskResume         | 恢复任务   |
| vTaskStartScheduler | 启动调度器 |

------

# 2️⃣ 调度器 Scheduler（RTOS核心）

| 单词           | 含义        |
| -------------- | ----------- |
| scheduler      | 调度器      |
| context switch | 上下文切换  |
| preemptive     | 抢占式调度  |
| cooperative    | 协作式调度  |
| time slice     | 时间片      |
| yield          | 主动让出CPU |

------

# 3️⃣ 队列 Queue（任务通信核心）

| 单词      | 含义               |
| --------- | ------------------ |
| queue     | 队列（任务间通信） |
| send      | 发送数据           |
| receive   | 接收数据           |
| blocking  | 阻塞等待           |
| overwrite | 覆盖写入           |
| peek      | 偷看数据（不取出） |

------
