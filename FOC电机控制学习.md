# FOC算法学习

## 1.如何得到需要的三相PWM波(基于SPWM)

### 1.1. **FOC算法中的正弦信号**

FOC算法的核心思想是将电机的三相电流通过 **Clarke** 和 **Park** 变换转换到一个旋转坐标系（d-q 坐标系）中，从而使得电流和转矩控制变得更加直观和精确。在 FOC 中，通常我们会得到以下信号：

- **Id**（d轴电流）：控制磁场的电流分量。
- **Iq**（q轴电流）：控制转矩的电流分量。

通过调节这两个电流分量，你可以控制电机的磁场（`Id`）和转矩（`Iq`）。通常 **`Iq`** 电流用于控制电机的转矩，而 **`Id`** 通常被设为 0（如果是永磁同步电机），避免产生有功功率。

这些电流信号（`Id` 和 `Iq`）通常会通过 **PID 控制器** 来调节，最终得到目标电压值，然后这些电压值通常会转换为 **三相电流信号**。

### 1.2. **利用硬件定时器生成固定的PWM(载波)**

载波在 PWM 信号的生成过程中扮演着非常重要的角色。载波的主要作用是定义 PWM 信号的频率和周期，它通常具有以下几个特点：

- **载波频率（`f_carrier`）** 通常高于参考信号（正弦波）的频率，典型值在几千赫兹到几十千赫兹之间。载波的频率决定了 PWM 信号的开关频率。
- **载波形状** 可以选择不同的波形，最常见的有 **三角波** 和 **锯齿波**。

载波和参考信号进行比较，产生一个在每个周期内高电平和低电平交替的 PWM 信号。具体来说：

- 载波波形通常是 **周期性变化** 的，它的幅度通常为固定值。
- 参考信号（正弦波）控制电压的变化，通过它和载波进行比较，从而决定占空比。

### 1.3. **为什么选择三角波作为载波？**

通常在 **FOC（矢量控制）** 中生成 PWM 信号时，我们选择 **三角波** 作为载波的原因主要是基于它的数学性质和硬件实现的简单性。三角波与参考信号的比较有以下优点：

- **线性变化**：三角波具有均匀的上升和下降时间，调制过程更为平滑，容易生成高质量的 PWM 信号。
- **频谱特性**：三角波的频谱比较简单，通常不会产生过多的高频噪声，这对于电机控制系统非常有益，尤其是对于电机驱动系统的电磁兼容性（EMC）有帮助。
- **实现简单**：三角波生成非常简单，且可以通过数字控制直接在硬件中实现。

#### 1.3.1. 如何利用硬件定时器得到三角波

​	**利用高级定时器的中心对齐** : 计数模式 = 向上/向下计数，定时器的 **自动重载值（ARR）** 决定了载波周期，这里的计数器(**`CNT`**)就是一个高度对称的三角波，硬件直接生成，无需CPU干预，高效

### 1.4. **PWM 信号的生成过程**

当你使用 **FOC 算法** 得到电机的 `Id` 和 `Iq` 信号后，你通常会通过 **逆 Clarke 变换** 和 **逆 Park 变换** 来将这些电流分量转换回三相电压或电流信号。这个过程的关键步骤是：

- **计算目标电压**：通过控制 `Id` 和 `Iq`，你会得到一个目标电压，这个目标电压会被转换为三相电压（通常是正弦波形）。
- **将目标电压与载波信号进行比较**：通过将目标电压信号与固定频率的载波信号（通常是三角波）进行比较，可以产生 PWM 信号。PWM 信号的频率是由载波决定的，占空比则由参考信号（如目标电压）决定。

### 1.5. **载波与参考信号的比较**

具体来说，PWM 信号的生成步骤如下：

1. **载波生成**：生成一个固定频率的载波信号，通常为三角波或锯齿波。载波频率 `f_carrier` 通常比电机控制频率高很多，例如 20kHz 到 50kHz 之间。
2. **参考信号（正弦波）生成**：根据 FOC 算法的输出（通常是目标电压信号），生成一个正弦波形。该波形的幅度和频率对应于电机控制的需求。
3. **比较**：将正弦波（参考信号）与三角波（载波信号）进行比较：
   - 如果参考信号（正弦波）的瞬时值大于载波的值，则输出高电平（1）。
   - 如果参考信号（正弦波）的瞬时值小于载波的值，则输出低电平（0）。
4. **PWM输出**：通过不断地比较，生成一个可调占空比的 PWM 信号。这个 PWM 信号的占空比随着参考信号（正弦波）幅度的变化而变化，从而控制电机的功率输出。

#### 1.5.1.硬件实现思路：

​	**配置定时器为 中心对齐模式（计数模式 = 向上/向下计数）。 定时器的 自动重载值（ARR） 决定了载波周期。 将 正弦调制值（经缩放）写入 CCR（捕获/比较寄存器）。 硬件自动比较 CNT 与 CCR，输出 PWM。**

​	**利用硬件优势，如果不利用比较器，则需要CPU参与，效率会降低。**

## 2.SVPWM(效率更高)

**概念：通过控制三相逆变器的开关状态，在电机定子上合成一个旋转的电压空间矢量 *Vref* ，使其尽可能逼近理想的圆形轨迹。**

### 2.1.clark变换(以及等幅值变换)和park变换

**此变换和下面的参考矢量计算为上下游关系，首先需要通过坐标变换来得到目标的参考矢量位置，然后由下游进行计算得到三相PWM** 

#### 2.1.1.clark变换

**本质是将三维进行降维操作，简化计算 ,从三相坐标降低为 二维坐标，<u>转换后的坐标系固定在定子上面</u>**

**clark变换将三相变换问题转换到一个二维坐标轴上($ \mathbf{\alpha(横轴)和\beta(纵轴)坐标系} $)，这个坐标轴固定在定子上面，现在还需要引入另外一个坐标轴(D Q坐标系)，此坐标系固定在转子上面**

**左边为定子的坐标轴，右边为转子的坐标轴**

**公式推导** ：

- **Park正变换：**

$$
\begin{bmatrix}
i_d \\
i_q
\end{bmatrix}
=
\begin{bmatrix}
\cos \theta & \sin \theta \\
-\sin \theta & \cos \theta
\end{bmatrix}
\begin{bmatrix}
i_a \\
i_b
\end{bmatrix}
$$

- **Park逆变换：**

$$
\begin{bmatrix}
i_a \\
i_b
\end{bmatrix}
=
\begin{bmatrix}
\cos \theta & \sin \theta \\
-\sin \theta & \cos \theta
\end{bmatrix}^{-1}
\begin{bmatrix}
i_d \\
i_q
\end{bmatrix}
$$

- **拆分：**

$$
i_a = i_d \cos \theta - i_q \sin \theta
$$

$$
i_b = i_q \cos \theta + i_d \sin \theta
$$



**D Q坐标系的D轴和$ \mathbf{\alpha(横轴)和\beta(纵轴)坐标系} $ 中的 $ I_{\alpha}$ 中间的夹角θ为<u>电角度</u>** 

- **电角度：**转子转动导致的D轴与 $I_{\alpha}$ 轴的角度为电角度
- **电角度和机械角度转换公式：**$\Large θ_e = p \times θ_m $ 
  - $ θ_e $ ： 电角度
  - $ θ_n $ :    机械角度
  - P :       电机极**对数**



### 2.2.电机上电调零

**<u>需要让电角度和磁编码器的机械角度形成对应关系，进行零位对齐</u>**

- **当电机上电的时候通常你并不知道现在的电角度位置，可以通过磁编码器得到机械角度，但是二者的零位并没有对应，没有进行调零**

#### 2.2.1.**调零步骤以及误差优化：**

1. 首先来确保编码器旋转方向和磁编码器一致，后面施加定向磁场观察旋转方向和编码器的数据增长方向，**如果方向不一致**，即给给定正向磁场 ($\large I_d >0$ )，观察编码器数值（0-360°）却在下降，此时方向取反，那么**直接对编码器数据做取反动作**

   - ```c
     // 假设磁编码器为 0- 65535 实际就是一共65536个数
     const uint32_t ENCODER_MAX = 65536U;   
     
     // 计算缩放到360°的缩放因子
     const float SCALE = 360.0f / (float)ENCODER_MAX;
     
     // 步骤1: 取反方向（使正转时角度增加）
     uint32_t inverted = ENCODER_MAX - (uint32_t)raw_angle;
     
     // 步骤2: 映射到 0~360.0
     float theta_deg = (float)inverted * SCALE;
     
     // 步骤3: 归一化到 [0, 360)
     theta_deg = fmodf(theta_deg, 360.0f);
     ```

     

2. 首先给定一个初始电角度值：$\theta_e = 0°$  然后**施加定向磁场**（$I_q = 0 \qquad I_d ≠ 0 $），定子产生的旋转磁场会推动转子转动，直到转子磁场与定子磁场对齐，此时静止下来就是来到了零点(如果电机无法锁住大概率是电压不够导致定向磁场没有力)

3. 可以用手摆动电机在不同极性位置锁定，找到偏移量的最小绝对值，此时为零点，选择读取磁编码器数值，**磁编码器** 会读取到当前的 **机械角度**，并且该位置可以作为电机的 **参考零位**。

4. 得到电角度和机械角度的**偏移量**，，配合原来本身的角度转化公式可以推导一个 **新的公式    ：**$\Large \theta_e  = p \times (\theta_m-\theta_{offset})  $ 

   - $\Large \theta_e$   : 电角度

   - $\Large \theta_m$ : 机械角度 

   - $\Large \theta_{offset}$ : 偏移量，有正负的
     - 如果偏移量是 **正值**，则表示机械角度 **大于** 电角度，电角度需要**减去偏移量**来进行补偿。
     - 如果偏移量是 **负值**，则表示机械角度 **小于** 电角度，电角度需要**加上偏移量**来进行补偿。

   - **偏移量误差重点！！！！！** ： <u>这里得到的偏移量仍旧有误差，需要进行校正</u> 
     - **方法1(适用于开环结构)： **需要通过让电机进行开环的正反转，观察速度差值，来进行手动微调(效率较低)
     - **方法2(适用于开环结构)：** 或者说通过延长对零时间，先等待电流完全稳定（如 500ms~1s），再多次采样平均，连续读 100 次编码器取中位数
     - **方法3(适用于闭环结构)：** **快速校正（基于 Id 最小化） ，初始待机状态（$I_q ≠ 0(施加电流激励) \qquad I_d = 0(理想情况下为0)$ ），通过瞬间注入 短暂 Iq 脉冲 ，在进入闭环后 10~50ms 内，施加一个小幅度、短时间的 $\large I_q$（如 10% 额定电流，持续 20ms），同时监测 $\large I_d$ 响应，理想 $\large I_d$ 应该等于 0 。**
       - **利用现有公式：$\large \Delta \theta\approx -\frac{I_d}{I_{q0}}$   ，（单位：弧度）其中 $\large I_{q0}$ 为设置的短暂电流脉冲，$\large I_d$ 为时间内的平均值**
       - 基**于原来调零的偏移量进一步减少误差得到我<u>最终的公式：</u> $\Large \theta_e  = p \times (\theta_m-\theta_{offset} ) +  \theta_{corr} $**
         - **$\large \theta_{corr}$ :进一步计算出的角度偏移值，原公式里面用的是弧度值，所以计算后需要转换，为弧度制转化为欧拉角后的角度，这种更为直观，这里暂时统一为角度制，如有需要，转换一下即可**



### 2.3.电流矢量（$I_q，I_d$）和电压矢量（$U_q，U_d$）的转换：

#### 2.3.1.数学模型转换作为电流闭环前馈

$$
\Large U_d = R i_d + L \frac{d i_d}{dt} - \omega L i_q + \lambda_m
$$

$$
\Large U_q = R i_q + L \frac{d i_q}{dt} + \omega L i_d
$$

- $U_d, U_q$: 分别是 $d$ 轴和 $q$ 轴的电压分量
- $i_d, i_q$: 分别是 $d$ 轴和 $q$ 轴的电流分量
- $R$: 电阻
- $L$: 电感（$d$ 轴和 $q$ 轴电感相同或不同，取决于电机的设计）
- $\omega$: 转速（旋转速度）
- $\lambda_m$: 磁链（通常由永磁体提供，是常量）

**缺少磁链参数可以进行模型简化：**
$$
\Large U_d = R i_d + L \frac{d i_d}{dt}
$$

$$
\Large U_q = R i_q + L \frac{d i_q}{dt} 
$$

#### 2.3.2.闭环结构，直接使用PI控制

**理论上通过电流反馈，然后经过PI控制器输出为电压矢量，这样就只需要给定参考电流矢量和已知反馈的电流矢量进行反馈闭环控制**







### 2.4.矢量空间的参考矢量计算：

**通过相邻的两个基准电压矢量来合成参考电压矢量**

**以六步换相的六个扇区划分，红线为转子需要锁定的位置，电压为 $ V^*$ ，然后列出第一行公式，用矢量合成的方式得到的关系，这里的 $ T_s $为一个周期，红色转子的上方一根线为 $ T_2 $ ,下面为 $  T_1 $** 

- **计算步骤：**

$ \begin{equation}
\Large |V^*| = \frac{|\frac{T_2 }{T_s} \times V_2|}{\sin \frac{2}{3} \pi} = \frac{|\frac{T_1}{T_s} \times V_1|}{\sin \left( \frac{\pi}{3} - \theta \right)}
\end{equation}$

$\begin{equation}
\Large T_s |V^*| = \frac{T_2 \cdot 2/3 \cdot V_{DC}}{3 \sin \theta} = \frac{T_1 \cdot 2/3 \cdot V_{DC}}{\sin \left( \frac{\pi}{3} - \theta \right)}
\end{equation}$

$\begin{equation}
\Large T_s |V^*| = \frac{T_2 V_{DC}}{3 \sin \theta} = \frac{T_1 V_{DC}}{\sqrt{3} \sin \left( \frac{\pi}{3} - \theta \right)}
\end{equation}$

$\begin{equation}
\Large T_s |V^*| = \frac{T_2 V_{DC}}{\sqrt{3} \sin \theta} = \frac{T_1 V_{DC}}{\sqrt{3} \sin \left( \frac{\pi}{3} - \theta \right)}
\end{equation}$

$\begin{equation}
\Large T_1 = \frac{\sqrt{3} |V^*|}{V_{DC}} T_s \sin \left( \frac{\pi}{3} - \theta \right)
\end{equation}$

$\begin{equation}
\Large T_2 = \frac{\sqrt{3} |V^*|}{V_{DC}} T_s \sin \theta
\end{equation}$

$\begin{equation}
\Large T_{0,7} = T_s - T_1 - T_2
\end{equation}$

### 2.5.基准电压矢量的幅值与SVPWM 的矢量空间转换

- 电压矢量刚好触碰到六边形边界等于它的最大输出能力，即六步换相里面相隔60°的基准矢量，如果需要运用到SVPWM这种矢量空间里面需要转换一下，即理想的旋转轨迹，为这个多边形的内切圆，被切成6个等边三角形，所以它的高为 $\frac{\mathbf2}{\mathbf3} \times \mathbf a$ ，a为边长，即基准矢量乘以一个系数就可以换算到SVPWM矢量空间里面来进行计算。

  - 为什么一定是内切圆而不是外接圆，因为你需要的矢量实际上都是由各个基准矢量合成的，但是一旦超过内切圆，它就会出现一些无法达到的区域，此时波形生成失败

  - **T1 和 T2**：这两个时间分配是通过对参考电压矢量 **V∗** 和六个扇区内的基准矢量的关系进行计算得出的。
    - 那么**T1** 和 **T2** 和<u>母线电压</u>(**在三相逆变器中，当某一相上桥臂导通、另一相下桥臂导通时，相间电压为 *VDC*** )的关系：
    - 所以 $ V_2和V_1 = \frac{2}{3} \times V_{DC} $

### 2.6.零矢量在SVPWM中的作用

**重点：这里为逆变器输出策略，与实际物理转子位置控制不一致** 

1. **补足 PWM 周期时间**
   - 任意参考矢量的合成时间满足：
     $ T_a + T_b + T_c = T_{pwm} $ 
   - 当 ∣*V**re**f*∣<*V*max 时，$ T_a + T_b  < T_{pwm} $  ，**必须用零矢量填充剩余时间，即低电平的时间**。
2. **实现电压矢量的时间平均合成**
   - 零矢量本身在 αβ 平面上为 (0, 0)，不影响合成矢量的方向和幅值，但保证了调制的完整性。
3. **控制共模电压与 EMI**
   - U₀（000）：中性点电压 ≈ 0
   - U₇（111）：中性点电压 ≈ $ V_{DC} $ 
   - 合理分配两者比例，可降低共模电压波动，减少电磁干扰（EMI）。
4. **支持零电压输出状态**
   - 当 *V**re**f*=0 （如停机、零转矩指令），**全部 PWM 周期输出零矢量**，实现安全静默控制。



### 2.7.如何得到三相PWM(七段法)

**扇区判断**：

- 在SVPWM中，首先需要判断目标电压矢量 $ U_{ref} $ 位于哪个扇区。由于SVPWM是基于**空间矢量**的，每个扇区的电压矢量都有不同的组合规则。
- **如何判断**：根据目标电压矢量的角度来判断其所在的扇区。通常，SVPWM将平面划分为6个扇区，每个扇区的范围是60°（例如，0°到60°为第1扇区，60°到120°为第2扇区，依此类推）。
  - **传感器方法：**利用磁编码器直接获取机械角度(**需要角度归一化为0-360°**)，然后转化为电角度
    - **核心公式：电角度 = 机械角度 × 极对数**
  - **无传感器方法** ： 
- **过程**：根据目标电压矢量的角度和位置，选择相应的两个基准电压矢量，这些基准矢量将在后续的计算中用来合成目标电压矢量。

**初始时间分配**：

- 0到60°，为第一扇区，如果需要移动到第二扇区，60°到120°，计算的时候用 60°和120°的基准矢量作为相邻的基准矢量，然后合成参考电压矢量，得到相邻的电压基准矢量，T1和T2 以及 T3(零矢量持续时间)。

**计算各相的总高电平时间**

某相的高电平时间 = 该相在所有阶段中 “导通” 的时间总和：

- **A 相高电平时间（T_A）**：V₄阶段（T₁/2）+ V₆阶段（T₂/2）+ V₇阶段（T₇）+ V₆阶段（T₂/2）+ V₄阶段（T₁/2）= T₁ + T₂ + T₇
- **B 相高电平时间（T_B）**：V₄阶段（T₁/2）+ V₇阶段（T₇）+ V₄阶段（T₁/2）= T₁ + T₇
- **C 相高电平时间（T_C）**：V₆阶段（T₂/2）+ V₇阶段（T₇）+ V₆阶段（T₂/2）= T₂ + T₇

**验证时间总时长是否正确，七分法已经保证了波形的对称性，高电平时间+低电平时间刚好等于 T_s**

$ T_s =  T_A + T_0 = T_1 + T_2 + T_7 + T_0 = T_1 + T_2 + (T_s - T_1 - T_2) = T_s $

$ T_s =  T_B + T_2 +  T_0 = T_1 + T_2+ T_7 + T_0 = T_1 + T_2 + (T_s - T_1 - T_2) = T_s  $ 

#### 2.5.1.**<u>转换PWM信号：与三角载波进行比较：</u>**

- 配置定时器为 中心对齐模式（计数模式 = 向上和向下计数，pwm比较模式为2，即CNT ≥ CCR的时候输出高电平）。 
- 定时器的自动重载值（ARR） 决定了载波周期，并且由于对称分布，CCR和ARR的关系：**CCR*=*ARR×(1−*D*/2)**  
  - **`D`：占空比，单位0-1 **   

- 举例：使用STM32G4为主控，选择生成20KHZ的三角波，配置定时器，计算出来载波周期为50μs，$ T_s = 2 \times ARR/f_{clk} = 2 \times ARR/170MHZ = 50μs $

  - 得出ARR = 4250，假如 $ T_A = T_1 + T_3 + T_7 = 20μs + 15μs + 5μs = 40μs , T_A / T_s = 40/50 = 0.8 = D $

  - $ CCR = 4250 \times (1 - 0.8/2) = 2550 $ 
  - **向上计数：`CNT ≥ 2550` 时高电平，时长 `4250 - 2550 = 1700`（计数单位）。**
  - **向下计数：`CNT ≥ 2550` 时高电平，时长 `4250 - 2550 = 1700`（计数单位）。**
  - **总高电平占比 `(1700×2) / (4250×2) = 0.8`，完全符合预期**

### 2.8.整体算法思路

首先我需要已知的I_q 和 I_d，然后直接换算为电压，通过数学转换得到 V_q 和 V_d，再通过帕克逆变换得到V_α 和V_β，此时已经得到定子坐标系的电压矢量坐标，然后需要开始计算扇区位置，通过夹角判断扇区，开始计算相邻基准矢量的高电平时间，再根据七分法得到相邻两个基准电压矢量通电时间，通过三角载波得到三相PWM的实际输出
